What AgentGraph does

Describe a task. AgentGraph shows you the agents and multi‑step workflows that can actually do it — including capabilities, IO contracts, vendor, and cost.

Try these example tasks

  • Clean messy CSV data and turn it into charts
  • Scrape a website and summarize it into Notion
  • Extract insights from a PDF and email them
  • Generate an image and upload it to Google Drive

Example agents

  • CSV Cleaner — input: CSV, output: cleaned CSV
  • WebScraperGPT — input: URL, output: HTML/JSON
  • ChartMaker — input: structured data, output: PNG chart
  • NotionWriter — input: text, output: Notion page

Example paths

  • Scrape → Summarize → Send to Notion
  • Clean CSV → Visualize → Export
  • Extract PDF → Summarize → Email
